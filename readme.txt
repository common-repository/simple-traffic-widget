Simple traffic widget is a free WordPress plugin that shows in the public interface of your website or blog number of current online users, hit counter, page views or unique visitors for last day, week and month.

== Description ==
* Custom labels and title so you can translate in any language
* Use it as widget in any sidebar or footer
* Choose what traffic stats you want to display (page views, number of hits, unique visitors)

= How to use =
1. Drag to your sidebar the *Simple Traffic Widget* from 'Appearance > Widgets'. 
2. Configure settings, define labels and save.

= To do features =
* Display traffic stats in dashboard of wp admin interface
* Display all type of traffic stats in the widget public interface
* Choose different design layouts

== Installation ==

1. Download plugin .zip file
2. Unzip and upload the folder to `/wp-content/plugins/` directory.
3. Activate the plugin
4. Drag on your prefered sidebar and make configuration
